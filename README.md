Resources of information related to the **POLY-88** and **POLY-8813** system from **1976**.

The POLY-88 was a cassette-tape storage based Intel 8080 / S-100 bus based system.

The POLY-8813 was an expanded disk storage version of the POLY-88 (that could fit up to 3x 5.25" floppy disks).  The ROM provisioned for booting a DOS from the disk (FDOS or CP/M).

Both systems share CPU, RAM, and video cards.


BYBEE folder is submitted archive collection by Bob ByBee (an original contemporary "power user" of the **POLY-8813** system), as provided in Summer 2025.
This includes his PM.EXE (Polymorphic Systems) emulator that supports a -88 command line argument to be more POLY-88 compatible; the emulator supports a number of cassette and disk image formats).
